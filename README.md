# Prompt Market

Prompt Market is a web-based platform where users can discover, buy, and share high-quality AI prompts for tools like ChatGPT, Midjourney, and other generative AI systems.

It is built using Django and designed to be scalable, modern, and production-ready.

---

## Live Features

In this web application, users can:

- 🔍 Browse and search AI prompts
- 🧠 Discover prompts for different categories (coding, business, design, marketing, etc.)
- 📦 View detailed prompt descriptions
- ⭐ Save and manage favorite prompts (future feature)
- 💬 Upload and share their own prompts (future feature)
- 💰 Buy and sell premium prompts (future feature)
