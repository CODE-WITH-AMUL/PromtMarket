## About Prompt Market

Prompt Market is a modern web-based platform designed for discovering, exploring, and sharing high-quality AI prompts for powerful generative AI tools such as ChatGPT, Midjourney, Claude, Gemini, and more.

The platform helps users quickly find prompts tailored to their needs, including:

- Coding
- Business
- Design
- Content Creation
- Marketing
- Productivity
- AI Workflows

Prompt Market aims to become a centralized ecosystem where users can efficiently discover prompts that improve productivity and unlock the full capabilities of AI systems.

This project is currently in **Version 1**, which contains a limited prompt collection and core functionality.

Future versions will introduce larger prompt collections, advanced search capabilities, community features, and a complete AI prompt marketplace ecosystem.

Built using **Django**, the project is designed with scalability, maintainability, and production readiness in mind.

## Installation

Clone the repository:

```bash
git clone https://github.com/CODE-WITH-AMUL/PromtMarket.git
```

Move into the project:

```bash
cd PromptMarket
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the environment:

Windows:

```bash
venv\Scripts\activate
```

Mac/Linux:

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run migrations:

```bash
python manage.py migrate
```

Start the server:

```bash
python manage.py runserver
```
